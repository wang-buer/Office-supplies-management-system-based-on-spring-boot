# 基于springboot的办公用品管理系统

#### Description
基于springboot的办公用品管理系统拥有三种角色，分别为管理员、员工和财务人员
- 管理员：物品管理、物品类型管理、物品申请管理、用户管理、审核申请、借还登记管理、申请管理、数据统计等
- 员工：物品申请、查看申请状态、查看借还登记
- 财务人员：物品管理、类型管理、申请审核、申请状态管理、借还登记
